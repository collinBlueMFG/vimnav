# vimnav
autohotkey script to emulate vim motions/commands across windows desktop

current functionality:

normal, insert, visual modes

HJKL movement

WB word forward/back

enter number to repeat movement x times (up to 100)

ctrl+numpad 5 to suspend script

GUI popups to show mode and movement number

ctrl+f to exit to normal mode without sending ESC key as well (sends regular ctrl+f when in normal mode)


press : to open command box, supports:

w = save

q = alt+f4

wq = save, alt+f4

ctrl+4 is bound to open the styles menu in excel, change to whatever you want.
